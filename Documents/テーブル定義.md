# テーブル定義

このドキュメントでは、Hydra Browserで使用されるデータ構造（モデル）を定義します。

## 1. PaneConfig

ワークスペースに保存されるペイン構成データ。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意識別子 |
| url | string | 現在のURL |
| resolution | PaneResolution | 解像度設定 |
| scale | number | 表示スケール（パーセンテージ、10-200） |
| partition | PartitionId | セッションパーティション識別子（A-Z） |

## 2. PaneResolution

ペインの解像度設定。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| width | number | ビューポート幅（ピクセル） |
| height | number | ビューポート高さ（ピクセル） |
| presetName | string? | プリセットデバイス名（オプション） |

## 3. PaneState

ペインの実行時状態（PaneConfigを拡張）。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意識別子 |
| url | string | 現在のURL |
| resolution | PaneResolution | 解像度設定 |
| scale | number | 表示スケール（パーセンテージ） |
| partition | PartitionId | セッションパーティション識別子 |
| title | string | ページタイトル |
| consoleState | ConsoleState | コンソール状態（normal/warning/error） |
| isLoading | boolean | 読み込み状態 |
| canGoBack | boolean | 戻るナビゲーション可否 |
| canGoForward | boolean | 進むナビゲーション可否 |

## 4. WorkspaceConfig

workspaces.jsonに保存されるワークスペース構成。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意識別子 |
| name | string | ワークスペース名 |
| panes | PaneConfig[] | ペイン構成のリスト |

## 5. WindowState

window-state.jsonに保存されるウィンドウ状態。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| x | number? | ウィンドウX座標 |
| y | number? | ウィンドウY座標 |
| width | number | ウィンドウ幅 |
| height | number | ウィンドウ高さ |
| isMaximized | boolean | 最大化状態 |

## 6. AppSettings

settings.jsonに保存されるアプリケーション設定。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| downloadPath | string | ダウンロードディレクトリパス |
| activeWorkspaceId | string | 現在アクティブなワークスペースID |
| theme | AppTheme | テーマ設定（light/dark/system） |
| language | AppLanguage | 言語設定（ja/en） |

## 7. DownloadItem

ダウンロード項目の状態。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | string | 一意識別子 |
| filename | string | ダウンロードファイル名 |
| savePath | string | 完全な保存パス |
| url | string | ダウンロードURL |
| totalBytes | number | 合計ファイルサイズ |
| receivedBytes | number | ダウンロード済みバイト数 |
| state | DownloadState | ダウンロード状態 |
| startTime | number | ダウンロード開始タイムスタンプ |
| paneId | string? | ソースペインID（オプション） |

## 8. ResolutionPreset

解像度プリセット定義。

| フィールド | 型 | 説明 |
|-----------|-----|------|
| category | ResolutionCategory | カテゴリ（Mobile/Tablet/Desktop） |
| deviceName | string | デバイス名 |
| width | number | 画面幅 |
| height | number | 画面高さ |

## 列挙型

### PartitionId
`'A' | 'B' | 'C' | ... | 'Z'`

セッション分離のための26種類のパーティション識別子。

### ConsoleState
`'normal' | 'warning' | 'error'`

タイトルバーの色表示用コンソールメッセージ状態。

### DownloadState
`'progressing' | 'completed' | 'cancelled' | 'interrupted'`

ダウンロード進捗状態。

### AppTheme
`'light' | 'dark' | 'system'`

アプリケーションテーマ設定。

### AppLanguage
`'ja' | 'en'`

アプリケーション言語設定。

### ResolutionCategory
`'Mobile' | 'Tablet' | 'Desktop' | 'Custom'`

解像度プリセットカテゴリ。
